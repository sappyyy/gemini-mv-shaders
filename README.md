Experiments with Gemini

Contains several shaders with 
 - halation effect (previously missing in ReShade)
 - black bars filling in style of SVP
 - high-quality Shadows/Highlights recreation from Adobe Camera Raw
 - simple overlays for experimenting, contrast, etc
 - auto black-white levels (aka adaptive levels)
 - grain
 - advanced unsharp mask shader (improved unsharp.fx with added options)
 - something else, to be updated
